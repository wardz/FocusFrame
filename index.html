Tables

Events
FocusData events. List of events that you can register for. (If you want to modify the built in focus frame, hook the functions inside FocusFrame.lua instead.)

Fields:

  • UNIT_HEALTH_OR_POWER arg1=event or nil, arg2=unit or nil
  • UNIT_LEVEL arg1=event or nil, arg2=unit or nil
  • UNIT_AURA arg1=event or nil, arg2=unit or nil
  • UNIT_CLASSIFICATION_CHANGED
  • PLAYER_FLAGS_CHANGED
  • RAID_TARGET_UPDATE
  • FOCUS_UNITID_EXISTS arg1=event, arg2=unit
  • FOCUS_SET arg1=event, arg2=unit
  • FOCUS_CHANGED arg1=event, arg2=unit
  • FOCUS_CLEAR

Usage:

    Focus:OnEvent("EVENT_NAME", callbackFunc)

Focus:Call (func, arg1, arg2, arg3, arg4)
Use any unit function on focus target, i.e CastSpellByName.

Parameters:

  • func func function reference
  • arg1
  • arg2
  • arg3
  • arg4

Usage:

  • Focus:Call(CastSpellByName, "Fireball") -- Casts Fireball on focus target
  • Focus:Call(DropItemOnUnit); -- defaults to focus unit if no second arg given

Get border color for debuffs. Uses numeric indexes.

Parameters:

  • debuffType string e.g "magic" or "physical"

Focus:GetBuffs ()
Get table containing all buff data for focus. Should be ran in an OnUpdate script or OnEvent("UNIT_AURA")

Returns:

    table data or empty table
Focus:GetDebuffs ()
Get table containing all debuff data for focus. Should be ran in an OnUpdate script or OnEvent("UNIT_AURA")

Returns:

    table data or empty table
Focus:GetCast ()
Get cast data for focus. Should be ran in an OnUpdate script.

Returns:

  1. table FSPELLCASTINGCORE cast data
  2. number Current cast time
  3. number Max cast time
  4. number Spark position
  5. number Time left formatted

Or

    nil
Focus:GetReactionColors ()
Get UnitReactionColor for focus. (player only, not npc)

Returns:

  1. number r

  3. number b

Data

Focus:GetData ([key=nil])
Get specific focus data. If no key is specified, returns all the data. See SetFocusInfo() for list of data available.

Parameters:

  • key string (default nil)

Returns:

    data or empty table

Or

    nil

Usage:

    local lvl = Focus:GetData("unitLevel")
